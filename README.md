# yywiki

## Before

```
# brew install imagemagick
# npm install pm2 -g
```

## Dev Environment

### Mongo Service

> Help yourself to install mongo

```
# mongod --config /your/path/of/mongodb.conf --fork
```

### Front End Service

> Help yourself to install node(>6) and npm(>5)

```
# cd client/framework
# npm install
# cd ../application
# npm run dev
```

### Back End Service 

```
# cd server
# npm run dev
```

### Visiting the wiki

Open `http://127.0.0.1:3000` in your browser.
